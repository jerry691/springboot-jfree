spring-boot-jfreecharts-example

Example of a Spring Boot project with JFreeChart to generate images.

Pie Chart Bar Chart Line Chart

Pie Chart

Call the REST Service:

/chart/pie/{width}/{heigth}

Example

/chart/pie/800/600

Bar Chart

Call the REST Service:

/chart/bar/{width}/{heigth}

Example

/chart/bar/800/600

Line Chart

Call the REST Service:

/chart/line/{width}/{heigth}

Example

/chart/line/1000/600
